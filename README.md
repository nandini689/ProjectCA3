QuizzBuzz Website using MEAN STACK

how to run code:-

For FrontEnd :
$ git clone https://github.com/nandini689/ProjectCA3.git  
$ cd SecureQuiz
$ cd quiz
$ npm install
$ ng serve 

For Backend : 
$ git clone https://github.com/nandini689/ProjectCA3.git 
$ cd QuizyBackend
$ npm install
$ npm run dev
